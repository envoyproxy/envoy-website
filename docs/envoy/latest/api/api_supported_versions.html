Supported API versions¶

Envoy’s APIs follow a versioning scheme in which Envoy supports multiple major API versions at any point in time. The following versions are currently supported:

  • v3 xDS API (active). Envoy developers and operators are encouraged to be actively adopting and working with v3 xDS.

The following API versions are no longer supported by Envoy:

  • v1 xDS API. This was the legacy REST-JSON API that preceded the current Protobuf and dual REST/gRPC xDS APIs.

  • v2 xDS API. Support for this was removed in Q1 2021.
