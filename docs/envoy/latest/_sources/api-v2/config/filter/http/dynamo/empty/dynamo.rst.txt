AWS DynamoDB
============



https://aws.amazon.com/dynamodb/

AWS DynamoDB :ref:`configuration overview <config_http_filters_dynamo>`.

This extension does not have a structured configuration, `google.protobuf.Empty
<https://developers.google.com/protocol-buffers/docs/reference/google.protobuf#empty>`_ should be used
instead.


.. _extension_envoy.filters.http.dynamo:

This extension may be referenced by the qualified name *envoy.filters.http.dynamo*

.. note::
  

  This extension is not hardened and should only be used in deployments where both the downstream and upstream are trusted.

.. tip::
  This extension extends and can be used with the following extension category:


  - :ref:`envoy.filters.http <extension_category_envoy.filters.http>`
